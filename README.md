# Node.js Shop practice project

An e-commerce backend project built with Node.js and Express.

## Features
- Authentication
- Product Management
- Cart System
- Orders
- File Uploads
- Payment Integration

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS


## ▶️ Getting Started

Clone the project:

```bash
git clone https://github.com/tahadeh2010/nodejs-shop.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
